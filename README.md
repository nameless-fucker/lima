# Lima

Lima is a cli tool for brwsing  files, editing text etc. Simply, it's a shell for CLI systems. It contains bugs (cannot save file properly). The bottom menu isn't ready-to-use; wait for path 1.1.


## Setup

To start Lima just use './lima' command while on the main directory of the branch.
Also source file is included so you can modify and compile lima.


## Lang 

Lima uses C++ and ncurses
